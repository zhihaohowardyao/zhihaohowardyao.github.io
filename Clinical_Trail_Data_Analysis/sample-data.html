About the Author

A set of ADaM datasets are simulated to create a virtual clinical trial enviroment


ADSL

  • USUBJID: dynamic ITT with default sample size = 60
  • MARKFL1: integer samples from a uniform distribution
  • MARKFL2: samples from a binomial distribution (n = 1, p = 0.6)
  • MARKFL1B: derived from MARKFL1 based on a dynamic cutoff

Two biomarkers are simulated, MARKFL2 is a binary marker (such as Gender). MarkerFL1 is an underlying continuous measurement (e.g. Pulmonary vascular resistance (PVR) in cardiology). A cut-off value is imposed on MarkerFL1 to define a binary marker MarkFL1B for the targeted treatment. Try to change the cut-off of the companion diagnostic here:
